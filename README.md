# VideoRecognitionTracking
# Description
Real time object or face detection recognition and tracking in video. The Full end-to-end project. 

# Requirements
The code requires Caffe as well as the following libraries:
* Opencv 3.0 or later
* Cuda 7.5 or later
* eigen3
* VS2013 or later
